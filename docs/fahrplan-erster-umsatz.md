# Fahrplan: von jetzt bis zur ersten Überweisung

Konkrete, abhakbare To-do-Liste bis zum **ersten Geldeingang** des ersten Kunden.

**Annahmen:** Deutschland, du startest als Einzelperson/Kleinunternehmer, erster Kunde =
vergünstigtes Referenzprojekt, **Werbebudget zahlt der Kunde** direkt an Meta.

**Wichtig:** Das erste Geld kommt **nicht erst, wenn die Werbung Ergebnisse bringt**, sondern
schon **direkt nach der Unterschrift** über die Onboarding-/Einrichtungsgebühr. Das ist der
schnellste Weg zum ersten Umsatz.

> Hinweis: Das hier ist Orientierung, keine verbindliche Rechts-/Steuerberatung. Im Zweifel kurz
> mit Finanzamt oder Steuerberater klären.

---

## Block A — Rechtliches & Konten (sofort, parallel machbar)

- [ ] **Gewerbe anmelden** beim Gewerbeamt (online/vor Ort, ~20–60 €). Ohne das keine Rechnung.
- [ ] **Kleinunternehmer-Regelung** wählen (§19 UStG, wenn Umsatz < 22.000 €/Jahr → keine Umsatzsteuer, einfacher). Kommt im „Fragebogen zur steuerlichen Erfassung" vom Finanzamt.
- [ ] **Steuernummer** abwarten (vom Finanzamt nach der Anmeldung) — für die Rechnung nötig.
- [ ] **Meta Business Manager + Werbekonto** anlegen (business.facebook.com) + eine Facebook-Seite.
- [ ] **Bankkonto/IBAN** bereit (eigenes reicht am Anfang; sauberer: getrenntes Konto).

## Block B — Angebot & Preis festlegen

- [ ] **Preis für den ersten Kunden** festlegen. Vorschlag Referenzprojekt: **Einrichtungsgebühr 300–500 € + Monatsbeitrag 300–500 €**, monatlich kündbar. Werbebudget zahlt der Kunde **extra, direkt an Meta**.
- [ ] **1-Seiten-Angebot/Vertrag** (Leistung, Laufzeit, Preis, was der Kunde liefert). → Claude kann das aufsetzen.
- [ ] Details: `geschaeftsmodell.md`.

## Block C — Ersten Kunden gewinnen

- [ ] **Anrufen** — Skript `wissen/sales.md`, Liste `leads/klienten-leads.csv` (Start: Hoch-Prio-Küchenstudios).
- [ ] **Termin/Demo:** deine Live-Landingpage zeigen, Case-Study-Deal erklären.
- [ ] **Zusage + Vertrag** → **jetzt die Onboarding-Gebühr in Rechnung stellen** (siehe Block F).
- [ ] **Kapazität prüfen** vor Abschluss (kann der Betrieb mehr Termine bedienen?).

## Block D — Für den Kunden aufsetzen

- [ ] **Material holen:** Logo, echte Fotos, Bewertungen, Öffnungszeiten (Vorlage-Mail: `../templates/onboarding-email.md`).
- [ ] **Landingpage personalisieren** (`../templates/landingpage/`, seine Variante) → eigene Netlify-Seite (`landingpage-hosten.md`) + **Impressum/Datenschutz** ergänzen.
- [ ] **Meta-Anzeige einrichten** (auf seinem Werbekonto, du als Partner), Budget, Kampagne **live** (Ad-Texte: `../templates/ad-copy.md`).
- [ ] **Testlauf:** Formular → Anfrage-Mail → Anruf klappt.

## Block E — Liefern & Beweis

- [ ] Kampagne läuft → Anfragen kommen → **schnell anrufen** (Speed-to-Lead) → Termine → erste Abschlüsse.
- [ ] **Ergebnisse festhalten** (Anfragen/Termine) + **Testimonial** einsammeln (deine Case-Study).

## Block F — Geld auf dein Konto

- [ ] **Rechnung schreiben.** Pflichtangaben: dein Name + Adresse, Kunde, Rechnungsnummer, Datum,
      Leistung, Betrag, IBAN, **Steuernummer**, bei Kleinunternehmer der Satz
      „Gemäß §19 UStG wird keine Umsatzsteuer berechnet".
- [ ] **Onboarding-Gebühr** direkt nach Vertrag berechnen; **Monatsbeitrag** dann laufend (z. B. zum Monatsanfang).
- [ ] **Kunde überweist** → 🎉 **erstes Geld auf deinem Konto.**

---

## Zeitrahmen & Realismus

- Block A/B: wenige Tage.
- **Flaschenhals = Block C** (das erste „Ja") — können 5 oder 50 Anrufe sein. Dranbleiben.
- Sobald das Ja steht, ist das erste Geld über die Onboarding-Gebühr schnell da.

## Dein allererster nächster Schritt

**Gewerbe anmelden + Meta-Konto anlegen + mit dem Anrufen starten** (Block A und C laufen parallel).
Die Technik (Landingpage) steht ja schon.
