# Offer- & Ad-Mechanik

Der wichtigste Hebel (Playbook §3): **wenige bewährte Strukturen, immer wieder recyceln.** Nicht ständig neue Konzepte erfinden — 3–5 Templates rotieren über alle Klienten, getestet wird nur in **Variationen** (Bild, Headline, Hook), nicht in neuen Konzepten.

## Die 3–5 Offer-Strukturen (rotieren)

1. **Gratis-Offer (Haupt-Struktur)** — „Kostenlose 3D-Küchenplanung" / „Kostenlose Bad-Beratung". Senkt die Hürde maximal; Geld wird im Abschluss verdient. Standard für Küche & Bad.
2. **Giveaway** — z. B. „Gewinne ein Küchen-Upgrade / Geräte-Paket im Wert von X" oder Gutschein. Sammelt viele Leads, danach Nurture zum Planungstermin.
3. **Anker-Preis „X statt Y"** — konkrete Aktion mit sichtbarer Ersparnis/Wert (z. B. Planungspaket-Wert, Aktionsrabatt auf Montage). Achtung: seriös halten, keine unrealistischen Versprechen.
4. **Saisonale Aktion** — Frühjahrs-/Jahresend-Aktion, „nur im [Monat]". Dringlichkeit + Anlass.
5. **(Optional) Terminkontingent/Verknappung** — „nur X Planungstermine pro Woche frei".

Für den Start pro Klient: **Gratis-Offer als Hauptkampagne**, 1 Variante als Test daneben.

## Trust-Regeln

- **Bewertungsanzeigen nie 5,0/5 zeigen — maximal 4,9** (5,0 wirkt gekauft). Sterne bleiben voll dargestellt.
- Echte lokale Bilder/Referenzen des Klienten schlagen Stockfotos.
- Klarheit vor Cleverness: ein Offer, ein klarer CTA (Termin/Planung).

## Personalisierung (macht Massenproduktion möglich)

**Struktur bleibt identisch — nur personalisieren:** Studioname, echte Fotos/Referenzen, Ort/Region, Ansprechpartner. So läuft dieselbe LP-/Ad-Vorlage über beliebig viele Klienten mit minimalem Aufwand.

## Meta-Optimierung (Kurz — Details in tech-stack.md)

Nicht nur „Lead"-Event an Meta senden, sondern **Deeper-Funnel-Events** (Termin gebucht → erschienen → Kunde) via CAPI — dann optimiert Meta auf echte Abschlüsse statt bloße Formular-Ausfüller.
